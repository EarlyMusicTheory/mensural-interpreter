Help

To evaluate the interpreter results, please add quality or num/numbase, rule and duration in minims.

The quality of a note is added only in cases of ambiguity according (following the current mensuration). Please do not use e.g. "perfecta" for already perfect notes. To remove an erroneously set quality choose "none".

While quality is used for standard imperfection/alteration, num/numbase is used for cases of partial imperfection.
An example: Given the change in the longa's value from 6 semibreves to 5 semibreves, the correct values would be num="6" and numbase="5".

Make sure to have added your name and initials before saving (otherwise it breaks). Thanks!
